搭建hexo blog

2022-11-02 PV:

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。Hexo是一个基于 node.js的快速生成静态博客的开源框架,支持 Markdown和大多数 Octopress插件,一个命令即可部署到 Github页面、 Giteee、 Heroku等,强大的APl,可无限扩展,拥有数百个主题和插件。简单来说就是一个不用你写代码，就能搭建一套属于你自己的个人博客网站。另外Hero还提供了大量主题模版供用户下载。

一、 环境准备

windows: node, git
mac: xcode就可以了

二、hexo安装

1、安装hexo

安装完成可输入hexo -v查看版本

2、初始化hexo，新建存储博客的文件夹

3、进入文件夹，安装一下npm

4、启动服务站点

访问http://localhost:4000/ 至此hero就搭建好了。可以在本地访问了

三、 GitHub建站

1.建立一个仓库

github上新建一个仓库，命名为user.github.io, 建立完成后，在settings–> Code and automation/Pages 去设置依赖的branch， 也可以根据自己的需要设置自定义域名。

2、安装hexo上传插件

3、修改hexo配置文件指定仓库路径

在user.github.io项目中找到_config.yml配置文件， 设置git仓库路径以及分支。

4、推送站点到github

可以看到main分支已经有最新的代码了， 等待一会，个站的页面就更新了。

5、访问

https://user.github.io/

四、更换主题

1.找到主题

https://hexo.io/themes/ ， 找一个喜欢的主题，

2.下载主题

user.github.io项目–> themes文件夹

cd到 themes， 将主题代码拉下来

3.更改配置

在user.github.io项目中找到_config.yml配置文件， 设置主题。

需要更新主题时

五、 更新仓库

1.新建文章

2.新建页面

user.github.io项目–> source文件夹–> 新生成的md文件

在主题配置文件中更新页面配置，将页面路径联接到页面上去

user.github.io项目 –> themes文件夹 –> hiker(当前主题) –> _config.yml

3.更新仓库

更新有一会会的延迟， 过一会查看就有了。
